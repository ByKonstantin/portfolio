# Tech Stack — Portfolio Website

## Цель
Максимально лёгкий, быстрый и адаптивный сайт-портфолио продуктового дизайнера.
Минимум зависимостей, никаких тяжёлых фреймворков.

---

## Ядро

| Технология | Версия | Назначение |
|---|---|---|
| **Vanilla JS** (ES Modules) | ES2022+ | Вся логика без фреймворков |
| **HTML5** | — | Семантическая разметка |
| **CSS3** | — | Стили, анимации, адаптивность |
| **Vite** | latest | Бандлер, dev-сервер, оптимизация сборки |

> Никакого React / Vue / Angular — они избыточны для статичного портфолио.

---

## Стилизация

| Технология | Назначение |
|---|---|
| **CSS Custom Properties** (переменные) | Дизайн-токены: цвета, типографика, отступы |
| **CSS Grid + Flexbox** | Адаптивные лейауты без сторонних библиотек |
| **CSS Keyframes / Animations** | Глич-эффект для ASCII-арта, переходы |
| **CSS Scroll Snap** | Горизонтальный скролл галереи кейсов |
| **@media / container queries** | Адаптивность под все разрешения |

> Tailwind и любые CSS-фреймворки не используются — лишний вес и зависимость.

---

## Анимации и эффекты

| Технология | Назначение |
|---|---|
| **CSS Keyframes** | Глич-эффект смены ASCII-арта в hero |
| **Intersection Observer API** | Отслеживание скролла для sticky nav (нативный браузерный API) |
| **CSS Transitions** | Выезд боковой галереи, hover-эффекты |

> GSAP и Motion One не используются — задача решается нативными CSS-анимациями.

---

## Шрифты

| Шрифт | Формат | Размер | CSS-токен | Назначение |
|---|---|---|---|---|
| **Golos Text Medium** | WOFF2 | 25 KB | `--font-body` | Наборный текст, описания, ссылки, кнопки |
| **Tiny5 Regular** | WOFF2 | 30 KB | `--font-heading` | Заголовки |
| **Fira Mono Regular** | WOFF2 | 63 KB | `--font-ascii` | ASCII-графика, слэши, сепараторы, моноширинные элементы |

- Все шрифты self-hosted в `public/fonts/` (WOFF2, конвертированы из TTF)
- `@font-face` с `font-display: swap` — мгновенная отрисовка, подмена после загрузки
- `<link rel="preload">` для всех трёх — браузер начинает загрузку параллельно с CSS

## Производительность

- **Цель**: Lighthouse Performance ≥ 95
- **Self-hosted шрифты** (WOFF2, `font-display: swap`, preload) — см. раздел «Шрифты»
- **Инлайн критический CSS** — Vite выносит критические стили в `<head>`
- **Никаких аналитических скриптов** на старте (можно добавить позже)
- **HTTP/2 + gzip/brotli** — настраивается на стороне хостинга

---

## Браузерная поддержка

- Все современные браузеры (Chrome, Firefox, Safari, Edge) — последние 2 версии
- Мобильные браузеры: iOS Safari, Android Chrome
- **Без полифилов** — используем только нативные API с широкой поддержкой

## Секции сайта

1. **Hero** — заголовок + ASCII-арт с глич-переходами между картинками
2. **Case A** × 5 — лейаут с текстом, ссылками и превью; клик → боковая галерея горизонтальным скроллом
3. **Case B** × N — альтернативный лейаут с теми же элементами и галереей
4. **Sticky Nav** — появляется на всех экранах кроме hero при скролле вверх
